# **App Name**: Tecnorampa Pro-Store

## Core Features:

- Industrial Component Catalog: A high-conversion grid layout featuring primary products like dock lamps and signaling traffic lights with high-quality imagery.
- Dynamic Product Configuration: Interface to select technical variants such as protective cages or specific voltage ranges for industrial components.
- AI Logistics Assistant Tool: A smart tool that reasons about user inputs like warehouse bay size and usage to recommend the optimal set of accessories and products.
- Persistent Procurement Cart: A localized shopping cart that persists across sessions, allowing industrial buyers to manage complex orders of high-count parts.
- Secure Gateway Integration: End-to-end checkout flow prepared for Stripe and PayPal for immediate B2B transactions.
- Technical Spec Explorer: An interactive data-sheet section for each product including the 40-inch articulating arm specs and IP65 protection ratings.
- Sales Lead Database: Integration with a backend database to store orders and customer contact details for future commercial follow-up.

## Style Guidelines:

- Primary Color: Industrial Amber (#FFC300) to match corporate branding, chosen for visibility and contrast in an industrial context.
- Background Color: Deep Carbon Grey (#1A1C1E) to evoke a professional, solid engineering environment and make the products pop.
- Accent Color: Steel Slate (#708090) for non-critical buttons and UI borders to maintain a 'robust machine' aesthetic.
- Body and Headline font: 'Inter', selected for its modern, machined, and highly legible quality that suits industrial specifications.
- Bold, thick-line technical icons (2pt stroke) that represent mechanical concepts and logistical safety.
- Heavy-duty grid system with generous spacing and large cards that imply reliability and strength.
- Controlled, precise transitions using ease-in-out timing to simulate industrial hydraulic movements.
